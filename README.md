# Braun-Unsigned-Multiplier-using-GDI-Technology
This project presents the design and implementation of a Braun Unsigned Multiplier using Gate Diffusion Input (GDI) technology.  By leveraging Tanner EDA tools (S-Edit, T-Spice, L-Edit). The optimized layout highlights the advantages of GDI for low-power VLSI applications, making this design suitable for embedded systems, IoT, and portable devices.
